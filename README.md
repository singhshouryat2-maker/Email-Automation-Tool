# Email Automation Tool

A lightweight React-based email automation workflow builder and manager.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` in your browser.

## Features

- Create custom email automation flows
- Filter emails with advanced queries
- Apply automated actions (label, forward, archive, etc.)
- Monitor workflow performance
- Instant and scheduled automation triggers
